## 测试数据策略

### 测试文件夹命名
- `TEST_MCP_001` - 用于文件夹操作测试
- `TEST_MCP_002` - 用于邮件移动测试
- `TEST_MCP_TRASH` - 用于删除测试

### 测试邮件
- 使用现有的邮件进行只读测试
- 复制邮件到测试文件夹进行修改测试
- 测试完成后删除测试文件夹

### 清理规则
1. 所有以 `TEST_MCP_` 开头的文件夹在测试结束后删除
2. 不删除真实邮件，只修改测试副本
3. 恢复测试邮件的原始状态

## 测试执行

将启动 4 个子 Agent 并行测试：
- Agent A: 连接 + 文件夹操作 (功能 1-2)
- Agent B: 邮件搜索 + 获取 (功能 3-4)
- Agent C: 标记操作 (功能 5)
- Agent D: 移动/复制 + MCP 接口 (功能 6-7)
